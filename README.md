# Task-04: Sentiment Analysis

## Objective
To analyze sentiment patterns in product review data using natural language processing techniques.

## Dataset
Product reviews dataset containing review text and sentiment labels.

## Steps Performed
- Loaded and inspected the dataset  
- Cleaned and preprocessed text data  
- Converted text into numerical features using TF-IDF  
- Trained a logistic regression classifier  
- Evaluated model performance  
- Visualized confusion matrix  

## Tools Used
- Python  
- Pandas  
- Scikit-learn  
- NLTK  
- Matplotlib  
- Seaborn  

## Outcome
This task demonstrates how NLP techniques can be applied to classify text data and extract sentiment insights.
